# 🏦 **BFSI Credit Risk: An End-to-End Machine Learning Pipeline**

An end-to-end Machine Learning solution designed to predict loan default risks and analyze model performance using industry-standard banking metrics. This dashboard allows financial institutions to move from "black-box" models to transparent, data-driven decision-making.

## 🚀 Project Overview

This project provides a comprehensive web-based dashboard for credit risk management:

* **Individual Risk Prediction:** Real-time probability of default calculations for loan applicants.
* **Multi-Model Engine:** Leverages a stack of four algorithms: **XGBoost, Random Forest, Decision Tree,** and **Logistic Regression**.
* **Banking Performance Metrics:** Detailed analysis using **KS (Kolmogorov-Smirnov) Tables**, **Confusion Matrices**, and **ROC AUC** scores.
* **Feature Explainability:** Visualizes the key drivers (e.g., Credit Score, DTI, Income) that influence the model's risk assessment.

## 📊 Model Comparison & Analytics

### **The Model Stack**

1. **XGBoost:** High-performance gradient boosting for maximum predictive power.
2. **Random Forest:** Ensemble of trees to ensure robustness and reduce overfitting.
3. **Decision Tree:** Provides a clear, interpretable logic path for credit decisions.
4. **Logistic Regression:** The statistical baseline for traditional credit scoring.

### **Key Metrics**

* **KS Statistic:** Measures the model's ability to discriminate between "Good" and "Bad" borrowers. A **Max KS > 40** indicates a high-performing credit model.
* **Confusion Matrix:** Evaluates the trade-off between "False Alarms" and "Missed Defaults"—the latter being the primary cost driver in banking.
* **Feature Importance:** Identifies which variables (e.g., **Credit History Score**) most significantly impact an applicant's risk profile.

## 🛠️ Installation & Setup

1. **Clone the directory** and ensure you have Python 3.10+ installed.
2. **Install dependencies**:
```bash
pip install -r requirements.txt

```


3. **Train the pipeline**:
```bash
python train_model.py

```


4. **Launch the dashboard**:
```bash
streamlit run app.py

```
